# fast-browser

A local AI agent that completes tasks in your active Chrome tab.

→ **Source:** [github.com/JayantDeveloper/fast-browser](https://github.com/JayantDeveloper/fast-browser)
→ **Privacy policy:** [/privacy](privacy.html)
→ **License:** MIT

## What it is

A Chrome side-panel extension and a CLI that share one agent core.
You bring your own LLM API key (Anthropic, Google Gemini, OpenRouter);
the agent runs entirely in your browser or on your machine.

## Receipts

- W3Schools 40-question HTML quiz: **40 / 40 in 4m44s, $0.31** (Haiku 4.5)
- Median agent step: **~785ms**
- Hacker News / Wikipedia tasks: 1 step, sub-cent

## Privacy

No telemetry. No analytics. No remote server. Page data is sent only
to the LLM provider you configured, only while a task you started is
running. See the [full privacy policy](privacy.html).
